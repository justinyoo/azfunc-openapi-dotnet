# Azure Functions OpenAPI Example for .NET #

This is the sample code repository for Azure Functions apps with the OpenAPI extension enabled in both .NET Core 3.1, .NET 5 and .NET 6. The sample code structure is as follows:

* [.NET Core 3.1 IoC](./NetCoreApp31.FunctionApp.IoC)
* [.NET Core 3.1 Static](./NetCoreApp31.FunctionApp.Static)
* [.NET 5 IoC](./Net50.FunctionApp.IoC)
* [.NET 5 Static](./Net50.FunctionApp.Static)
* [.NET 6 IoC (in-proc)](./Net60.FunctionApp.InProc.IoC)
* [.NET 6 Static (in-proc)](./Net60.FunctionApp.InProc.Static)
* [.NET 6 IoC (out-of-proc)](./Net60.FunctionApp.OutOfProc.IoC)
* [.NET 6 Static (out-of-proc)](./Net60.FunctionApp.OutOfProc.Static)
